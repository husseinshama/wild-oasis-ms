

# 🏔️ The Wild Oasis 

A professional, full-featured internal management dashboard designed for a luxury boutique hotel and resort. This application allows hotel staff to efficiently manage cabins, bookings, guests, and check-in/check-out statuses, while providing real-time data analytics and performance statistics.

---

## 🚀 Tech Stack

- **Frontend:**React.js (Vite)
- **State Management & Data Fetching:** React Query (TanStack Query)
- **Backend & Database:** Supabase (PostgreSQL)
- **Styling:** Styled Components
- **Charts & Analytics:** Recharts
- **Routing:** React Router DOM

---

## 🌟 Key Features

- **Dynamic Dashboard:** Displays real-time summaries of sales, occupancy rates, and stay durations using interactive charts (Pie & Area charts).
- **Today's Activity:** A dedicated live feed for hotel staff to track today's scheduled arrivals and departures with seamless one-click Actions.
- **Bookings Management:** Comprehensive system to view, filter, sort, and update booking statuses, including breakfast add-ons and payment confirmations.
- **Cabins Management:** Full CRUD operations for creating, updating, copying, or deleting hotel cabins, complete with image upload capabilities.
- **Dark Mode Toggle:** Seamless light/dark mode adaptation that persists user preferences using Local Storage.
- **Authentication & Security:** Protected routes ensuring only authenticated hotel staff can access sensitive internal data.
